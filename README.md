# Universal Share

The UniversalShareInput component is a lightweight, client-side social sharing solution built using React. It leverages the react-share library to enable seamless sharing across multiple platforms from a single, unified interface.

This component is designed for easy integration within Mendix custom widgets and supports dynamic platform selection, validation handling, tooltips, and responsive UI behavior.
## Features
### •	url: 
The URL to be shared across platforms 
### •	title: 
Title or message used in supported platforms 
### •	description: 
Additional content for platforms like LinkedIn & Email 
### •	hashtags: 
Comma-separated hashtags for supported platforms 
### •	
imageUrl: Required for platforms like Pinterest 
### 	platforms: 
Array of selected social platforms to display 
### •	appid: 
Required for Messenger sharing
